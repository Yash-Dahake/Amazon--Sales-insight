# Amazon--Sales-insight

## 📌 Project Title
Amazon Year-wise Sales Dashboard

Interactive Power BI dashboard showcasing Amazon’s year-wise sales (2012–2015). Features include KPI metrics, segment and market sales breakdowns, regional insights via map, customer profit analysis, and top/bottom product profitability, enabling clear performance tracking and decision-making.


## 📝 Short Description
Interactive Power BI dashboard analyzing Amazon’s sales performance from 2012 to 2015.  
It highlights KPIs, segment and market breakdowns, regional insights, and product profitability.

## 🎯 Purpose
The purpose of this project is to provide clear insights into Amazon’s yearly sales trends, customer profitability, and product performance.  
It helps stakeholders identify growth opportunities, track KPIs, and make data-driven decisions.

## 🛠️ Tech Stack / Tools Used
- **Microsoft Power BI** – for dashboard creation and visualization  
- **Microsoft Excel / CSV** – for data preparation and cleaning  

## 📊 Data Source
The dataset is based on Amazon’s sales records (sample dataset).  
It includes yearly sales figures, customer details, product categories, and regional market information.

## 🌟 Features & Insights
- **Year Selection Tabs**: Switch between 2012–2015 to view metrics.  
- **KPI Metrics**: Sales projection, product units, KPI ratings, and returns.  
- **Sales by Segment**: Consumer, Corporate, and Home Office breakdown.  
- **Sales by Market**: Regional distribution across Asia Pacific, Europe, USCA, LATAM, and Africa.  
- **Geographical Insights**: World map showing sales distribution by region.  
- **Customer Profit Analysis**: Top customers ranked by profit contribution.  
- **Product Profitability**: Top 5 and Bottom 5 products by profit.
